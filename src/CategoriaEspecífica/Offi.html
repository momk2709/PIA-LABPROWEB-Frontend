Offimática

Programa de formación que busca enseñar a las personas el uso eficiente de las herramientas informáticas para el procesamiento y gestión de información, principalmente en un ambiente laboral. Estos cursos se enfocan en enseñar habilidades prácticas y específicas relacionadas con programas de software como Microsoft Word, Excel, PowerPoint, Access, entre otros.

Microsoft Word Avanzado

(20 hrs)
Microsoft Excel Básico

(15 hrs)
Microsoft PowerPoint Intermedio

(12 hrs)
Microsoft Access Fundamentos

(18 hrs)

Curso de ofimática sirve para proporcionar a los participantes las habilidades necesarias para utilizar eficientemente las herramientas de software de oficina, lo que les ayuda a ser más productivos, comunicativos y competitivos en el entorno laboral.
